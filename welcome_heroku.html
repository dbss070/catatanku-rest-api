Here's url command for CREATE, READ, UPDATE, and DELETE catatan data :

- CREATE new catatan (POST* - form.urlencoded) : https://catatanku-zayd.herokuapp.com/catatan

- READ all catatan (GET - query) : https://catatanku-zayd.herokuapp.com/catatan/all

- READ single catatan by id (GET - query): https://catatanku-zayd.herokuapp.com/catatan?id=1

- UPDATE existing catatan by id (PUT* - query and form.urlencoded) : https://catatanku-zayd.herokuapp.com/catatan?id=1

- DELETE existing catatan by id (DELETE* - query) : https://catatanku-zayd.herokuapp.com/catatan?id=1


* use API client for testing tools like POSTMAN
